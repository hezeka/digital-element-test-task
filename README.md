# Тестовое задание для цифрового элемента

## Инструкция

```bash
# Установить зависимости
$ yarn install

# Запустить режим разработки на 3000 порту
$ yarn dev

# Создание билда
$ yarn build
$ yarn start

# Сгенерировать статический сайт
$ yarn generate
```
